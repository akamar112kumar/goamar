Hoisting

  • Function declarations are hoisted with definitions.
  • var is hoisted (initialized to undefined).
  • let/const are hoisted but in TDZ (Temporal Dead Zone) until declared.

Closure

A function remembers variables from its lexical scope even after the outer function has finished.

The this Keyword & Common Pitfalls

  • Global (non-strict): this === window in browsers
  • Strict mode: this is undefined in functions
  • Method call: obj.method() → this = obj
  • Event listener: this is element (non-arrow fn)
  • Arrow functions: no own this, they capture lexical this
Pitfall: losing this when passing methods as callbacks.

Event Loop (Microtasks vs Macrotasks)

  • Call stack runs synchronous code.
  • Microtasks: Promise.then, queueMicrotask, MutationObserver
  • Macrotasks: setTimeout, setInterval, DOM events, I/O
  • Order: after each stack empty, run all microtasks, then one macrotask.

Heap Memory (and Stack)

  • Stack: function calls, primitive local variables.
  • Heap: objects/functions/closures allocated here.
  • GC reclaims unreachable objects (mark-and-sweep). Avoid leaks via global refs, detached DOM, long-lived caches.

Callback

A function passed to another function to be executed later (async or sync).

Callback hell: nested callbacks become hard to manage → use Promises/async.

setTimeout & setInterval

Prefer setTimeout loop for precise intervals and to avoid drift.

for...of vs for...in

  • for...of: iterates values of iterables (arrays, strings, maps, sets)
  • for...in: iterates enumerable keys over objects (including inherited) — use cautiously

Promises

Why Introduced

To avoid “callback hell”, provide composable async flows, error propagation, and better control.

Single Promise

Nested Promise (avoid)

Promise Chaining (preferred)

Run Multiple in One Call

  • Promise.all: waits for all, rejects fast if any fail
  • Promise.allSettled: waits for all, never rejects (gives statuses)
  • Promise.race: first to settle (resolve or reject)
  • Promise.any: first to fulfill (rejects with AggregateError if all reject)
