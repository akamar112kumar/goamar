Data types in JavaScript are categorized into two main groups: primitives and objects.

These represent single, immutable values.

  • Number: For both integer and floating‑point numbers, including Infinity and NaN.
  • BigInt: For integers of arbitrary size.
  • String: For text enclosed in quotes.
  • Boolean: Logical values true or false.
  • Null: Represents intentional absence of a value.
  • Undefined: Variable declared but not assigned.
  • Symbol: Unique identifiers used as object keys.

Object: For more complex data structures, such as arrays, functions, and key-value pairs.
